Backend

A continuación, se muestra la lista de los componentes de software más importantes que han sido utilizados para diseñar e implementar Trapper:

  • Ubuntu Sistema operativo base en el que se desarrolló y probó Trapper.

  • PostgreSQL Sistema de gestión de bases de datos relacionales (RDBMS) de código abierto e industria estándar, con PostGIS como su extensión espacial.

  • nginx nginx [motor x] es un servidor HTTP y de proxy inverso, un servidor de correo electrónico y un proxy TCP genérico.

  • Gunicorn Esta aplicación puramente en Python se utiliza para servir el proyecto Trapper.

  • Celery Esta aplicación de Python se utiliza para ejecutar tareas de manera asíncrona, lo que permite a Trapper funcionar más rápido y eficiente. Celery se utiliza para generar miniaturas de archivos grandes o videos, o para procesar colecciones cargadas por los usuarios.

  • Django Componente central de Trapper, un marco web de alto nivel mantenido por la Fundación Django Software. El marco Django simplifica y acelera significativamente la creación de sitios web complejos con bases de datos y enfatiza la reutilización y la pluggabilidad de sus componentes. Además, se han utilizado una variedad de aplicaciones de terceros de código abierto de Django para desarrollar Trapper. Para obtener una lista completa de estas aplicaciones, consulte `este archivo <>`_.

  • uMAP uMap permite crear mapas con capas de OpenStreetMap en un minuto y incorporarlos en su sitio. Utiliza django-leaflet-storage y Leaflet.Storage, construidos sobre Django y Leaflet.

Frontend

La interfaz de usuario de Trapper se desarrolló con base en tres soluciones independientes:

  1. Plantillas HTML (versión 5) y sus estilos CSS (impulsados por SASS)

  2. Un conjunto de scripts escritos en JavaScript puro (ECMAscript 5).

  3. Todas las bibliotecas y marcos externos incluidos en el proyecto:

  • Twitter Bootstrap Bootstrap Sass Oficial, que es la versión SASS oficial de Twitter Bootstrap. Esta biblioteca proporciona un conjunto de componentes HTML y estilos CSS utilizados para crear el esqueleto de Trapper.

  • Font Awesome Font Awesome, webfont de iconos vectoriales utilizados en el proyecto.

  • Angular JS Angular JS. Todas las grillas/tablas, incluyendo sus filtros, se han creado encima de este marco de Google.

  • Angular Cookies Módulo oficial angular para la gestión de cookies.

  • Angular Sanitize Módulo oficial angular que mejora la unión de datos en plantillas de angular.

  • Moment Biblioteca extremadamente poderosa Moment para el análisis y manipulación de fechas.

  • Select2 Solución completa Select2 que extiende los controles select HTML por defecto.

  • Select2 Bootstrap CSS Estilos de CSS para Select2, para adaptar su estilo a Twitter Bootstrap.

  • Bootstrap WYSIHTML5 Plugin JavaScript que agrega un editor de texto WYSIWYG al table.

  • Bootstrap Datepicker http://eternicode.github.io/bootstrap-datepicker/ Widget de fecha simple.

  • Jquery Timepicker Widget Jquery que es solo un reloj.

  • Bootstrap Datetimepicker Widget de JavaScript que combina ambos reloj y fecha picker.

  • Video JS Esta biblioteca extiende a los jugadores de video HTML5 estándar.

  • Video JS Rangeslider Plugin de Video JS que permite establecer y obtener secuencias de video.
