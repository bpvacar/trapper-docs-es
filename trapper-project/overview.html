Resumen

TRAPPER es una aplicación web de próxima generación, basada en Django, que revoluciona la gestión de proyectos de trampa de cámaras! 🎯

A medida que la trampa de cámaras se vuelve cada vez más crucial para la investigación ecológica y la conservación de vida silvestre, el desafío de gestionar grandes conjuntos de datos multimedia aumenta exponencialmente. TRAPPER transforma este desafío en una oportunidad proporcionando una plataforma integral y potente basada en inteligencia artificial que maneja todo desde la ingesta de datos hasta análisis avanzados.

Lo que diferencia a TRAPPER es su enfoque moderno para la gestión de datos de trampa de cámaras: cumplimiento total con el estándar global emergente Camtrap DP, líneas de procesamiento de AI de vanguardia y flexibilidad de despliegue que se adapta desde la infraestructura de nube hasta la investigación en el campo. Ya sea que seas un investigador trabajando en un estudio de campo pequeño o gestiones proyectos de ciencia ciudadana a gran escala, TRAPPER se adapta a tus necesidades! 🚀

TRAPPER en un nutshell

  • 🌍 Estándares Globales: Soporte completo para el estándar emergente Camtrap DP

  • 🤖 Inteligencia Artificial: Procesamiento automático de datos con detección de objetos, clasificación de especies y protección de privacidad mediante inteligencia artificial

  • 🔧 Integración de AI: Integración fácil de nuevos modelos y algoritmos de IA

  • ⚡ Procesamiento escalable: Procesamiento de imágenes y videos basado en GPU que se adapta a tus necesidades

  • 🏗️ Despliegue flexible: Despliega donde sea - desde laptops hasta infraestructura de nube

  • ☁️ Listo para la Nube: Compatible con Azure, AWS, OVH, MinIO y más

  • 📱 Portátil: Funciona en servidores micro (Raspberry Pi 5, NVIDIA Jetson)

  • 👥 Interfaz dual: Módulo de expertos para investigadores + interfaz amigable para ciencia ciudadana

  • 🗄️ Gestión avanzada: Base de datos espacialmente habilitada con gestión de proyectos robusta

  • 🔄 Fuentes Abiertas: Completamente abierto con desarrollo colaborativo

Ecosistema TRAPPER

Trapper Expert — la aplicación web central (TRAPPER), que proporciona todas las funcionalidades del sistema esenciales. Cuenta con una interfaz frontend basada en Angular y Bootstrap, junto con un backend de Django. La aplicación utiliza una base de datos PostgreSQL con la extensión PostGIS para la gestión de datos espaciales, y emplea RabbitMQ como broker de mensajes con trabajadores Celery para el procesamiento asincrónico de tareas.

Trapper Citizen Science — una aplicación frontend innovadora (UI) para TRAPPER, diseñada específicamente para proyectos de ciencia ciudadana.

Trapper AI — un módulo responsable de la integración y configuración de modelos de IA para detección de objetos y clasificación de especies. Este componente incluye la Interfaz del Administrador de Trapper AI web y Trapper AI Worker. Los trabajadores pueden utilizar GPU y se pueden escalar o desplegar en varias máquinas.

Trapper Jupyter Hub (no aún lanzado) — un módulo para compartir JupyterHub y cuadernos de Jupyter dentro de TRAPPER, permitiendo a los usuarios explorar y analizar datos utilizando bibliotecas científicas de Python o R.

Trapper Tools — una aplicación CLI de Python para crear y subir paquetes de datos de trampa de cámaras compatibles con el sistema TRAPPER. Esta herramienta se recomienda para subir grandes volúmenes de datos (muchos GB) a la aplicación web TRAPPER. Cuenta con un pipeline automático que puede convertir, empaquetar y subir datos en una sola orden. Opcionalmente también puede disparar el procesamiento posterior a la carga de datos, incluyendo la construcción automática de secuencias y un pipeline de IA para detección de objetos y clasificación de especies tanto para imágenes como videos.
