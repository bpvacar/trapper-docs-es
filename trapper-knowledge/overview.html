Bienvenidos a la documentación del ecosistema TRAPPER

Esta es la central de documentación para el ecosistema TRAPPER — un conjunto modular de componentes que apoyan la investigación y la ciencia ciudadana en la monitorización de vida silvestre.

Visión general

El ecosistema TRAPPER consta de varios componentes interrelacionados:

  • Trapper Expert – Trapper Expert es la aplicación web central (TRAPPER), que abarca todas las funcionalidades esenciales del sistema. Cuenta con una interfaz frontend basada en Angular y Bootstrap, combinada con un backend basado en Django. La aplicación utiliza una base de datos PostgreSQL con extensión PostGIS para el manejo de datos espaciales, y emplea RabbitMQ como broker de mensajes con trabajadores Celery para el procesamiento asincrónico de tareas.

  • Trapper Citizen Science – Aplicación frontend innovadora e inteligente (UI) para TRAPPER, diseñada específicamente para proyectos de ciencia ciudadana.

  • Trapper AI Manager – Módulo responsable de la integración y configuración de modelos de inteligencia artificial para la detección de objetos y clasificación de especies. Este componente está organizado utilizando el componente web-based Trapper AI Manager y el trabajador Trapper AI Worker. Los trabajadores pueden utilizar GPUs y pueden escalarse o lanzarse en múltiples máquinas.

  • Trapper AI Worker – Módulo responsable de la integración y configuración de modelos de inteligencia artificial para la detección de objetos y clasificación de especies. Este componente está organizado utilizando el componente web-based Trapper AI Manager y el trabajador Trapper AI Worker. Los trabajadores pueden utilizar GPUs y pueden escalarse o lanzarse en múltiples máquinas.

Cada módulo puede ejecutarse de forma independiente, pero están diseñados para interaccionar en un ecosistema unificado.

Arquitectura y interacciones

El ecosistema se basa en microservicios que comunican entre sí mediante APIs REST y colas de mensajes:

  • Trapper Expert actúa como coordinador central, gestionando usuarios, metadatos y proyectos.

  • Trapper Citizen Science se comunica con la API REST del Expert.

  • Trapper AI Manager proporciona una API y panel de control para gestionar modelos y trabajadores.

  • Trapper AI Worker se conecta al AI Manager y procesa tareas utilizando Celery y RabbitMQ.
